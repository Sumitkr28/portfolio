## 🛠 Built with

`HTML` · `CSS` · `JavaScript` — no framework, no build step.
Canvas hero animation, scroll reveals, and a working contact form via [Web3Forms](https://web3forms.com).

## ✏️ Editing

All content (skills, projects, experience, certifications) lives in **[`data.js`](data.js)**.
Edit that file and refresh — no rebuild needed.

## ▶️ Run locally

```bash
npx http-server -p 8000 -c-1
# open http://localhost:8000
```
